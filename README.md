# Schedule Agent - 个人日程管理助手

基于 IMA notes API 的个人日程管理 skill，提供任务管理、习惯追踪和智能日记编写能力。

## 功能特性

- 📅 **月任务管理** - 管理月度任务计划，关注任务 ddl 和生命周期
- 📆 **周任务管理** - 细化周任务安排，关注任务在周内哪几天完成
- ✅ **日任务管理** - 每日任务列表，支持随时调整
- 📊 **习惯追踪** - 记录任务执行数据和用户习惯，提供个性化建议
- 📔 **智能日记** - 陪伴者视角的日记编写，周日自动生成周总结

## 文档结构

## 笔记本命名规范

| 功能模块 | 笔记本命名 |
|----------|-----------|
| 月/周任务 | `plan_{year}_{month}` |
| 日任务 | `task_{year}_{month}` |
| 用户习惯 | `habit` |
| 日记 | `diary_{year}_{month}` |

## 快速开始

1. 将 `schedule-agent` 目录复制到你的 IMA skills 目录
2. 在对话中唤起 skill："今天要做什么？"、"写今天的日记" 等
3. Skill 会自动创建对应的笔记本和笔记

## 设计原则

- **伴随式辅助** - 提供建议，但不强制
- **用户控制权** - 所有操作需要用户确认
- **按需创建** - 笔记本按需创建，避免空笔记本
- **永久保留** - 所有历史数据默认保留

## 依赖

- IMA notes API (`openapi/note/v1`)
- ima_cos_util（用于上传笔记内容）
